# Link

## Favicons

All favicons live under `favicons/` as plain local files referenced by the `favicon` column in `link2.csv` (e.g. `./favicons/github.com.png`).

There is no download script or build step. Favicons are fetched and placed manually (via Claude) when a link is added or a site's icon changes — edit `link2.csv` and drop the file into `favicons/` directly.
